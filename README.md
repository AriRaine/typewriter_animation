# Interface com Typewriter Effect

Banner com efeito de digitação animada, desenvolvido utilizando apenas HTML e CSS, com foco em experiência visual e animações.

---

## Funcionalidades

-  Efeito de digitação (typewriter)
-  Cursor animado piscando  
-  Animações sequenciais entre linhas  
-  Background com imagem e overlay escuro  
-  Layout responsivo

---

## Tecnologias

- HTML5  
- CSS3 (animações com `@keyframes`)

---

## Observação

Projeto desenvolvido para @TypoFX como prática de desenvolvimento frontend com foco em animações em CSS.
